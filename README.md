# 🏠 Property Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts residential property prices using Machine Learning techniques. The complete workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, dimensionality reduction using PCA, model training, hyperparameter tuning, and model evaluation.

---

## 🎯 Project Objectives

- Predict property prices accurately.
- Handle missing values and categorical features.
- Perform Exploratory Data Analysis (EDA).
- Apply Feature Engineering.
- Train and compare multiple Machine Learning models.
- Select the best-performing model.
